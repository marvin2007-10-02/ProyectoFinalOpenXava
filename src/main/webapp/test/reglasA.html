Vocabulario Forma A

Estas son las reglas que se aplican al subtest Vocabulario Forma A, codigo VOC1.

El test corresponde a Vocabulario Forma A, codigo VOC1.

El test tiene 37 preguntas.

Cada pregunta tiene cinco opciones: A, B, C, D y E.

El evaluado debe elegir el sinonimo correcto de la palabra presentada.

El tiempo limite es de 5 minutos.

Cada respuesta correcta suma 1 punto.

Las respuestas incorrectas no restan puntos.

Las preguntas no respondidas se consideran incorrectas, pero tampoco restan puntos.

El puntaje directo es la cantidad total de aciertos.

El percentil no representa porcentaje de respuestas correctas.

El percentil representa la posicion del evaluado segun las normas BFA.
